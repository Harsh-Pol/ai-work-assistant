# 🚀 AI Work Assistant

An AI-powered tool to automate everyday tasks like summarization, key point extraction, and email generation using Generative AI.

## 🔥 Features
- 📄 Upload PDF and extract content
- ✍️ Summarize long text instantly
- 📌 Extract key points in bullet format
- 📧 Generate professional emails

## 🛠️ Tech Stack
- Python
- Streamlit
- Google Gemini API
- PyPDF2

## 💡 How It Works
1. User enters text or uploads a PDF
2. Selects a task (summarize, extract, email)
3. AI processes input using LLM
4. Output is generated instantly
